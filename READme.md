# College Hackathon Frontend

## 📖 About the Project
This project is a frontend web application developed as part of my **2nd Semester College Hackathon**. It is designed to manage student requests such as On-Duty (OD) and Leave applications through different user roles.

## 🚀 Features
- Student Login
- Faculty Login
- Advisor Login
- HOD Login
- Principal Login
- Create New Account
- Role Selection
- On-Duty (OD) Request
- Leave Request
- Other Permission Request
- Simple and User-Friendly Interface

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript


## 🎯 Purpose
This project was created to improve my frontend development skills and to provide a simple interface for managing student permission requests.
